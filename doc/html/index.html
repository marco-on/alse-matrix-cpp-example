Ejemplo de Documentación de un programa para un sistema digital básico.

Documentación de programa en C++ para encender/apagar pines digitales de salida y leer el estado de los pines de entrada de un microcontrolador (MCU) ATmega328P en tarjeta Arduino UNO.

Descripción del programa.

Este programa muestra el funcionamiento de un sistema de cuatro (4) entradas (DS_IN_[3:0]) cuatro (4) salidas digitales (LED_[3:0]), donde el bit más significativo (MSb), tanto para el bit de entrada (DS_IN_3) como el de salida (LED_3), representa el modo de funcionamiento del sistema. Por tanto, el sistema tiene 2 modos de funcionamiento:

  • modo passthrough (1)
    El estado de las entradas DS_IN_[2:0] se ve reflejado en las salidas LED_[2:0].
  • modo encendido (0)
    Las salidas LED_[2:0] siempre están en estado activo sin importar el estado de las entradas DS_IN_[2:0].
